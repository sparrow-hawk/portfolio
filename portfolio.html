Python Discord 노래봇

Python의 discord.py 라이브러리를 사용해, 사용자가 음성 채널에서 명령어로 음악을 재생할 수 있는
Discord 노래봇을 개발 중입니다. YouTube API와 FFmpeg 등을 연동하여 스트리밍 기능을 구현했습니다.

mireaDB 연동 게시판 프로젝트

HTML/CSS와 PHP를 이용해 웹 게시판을 구현하고, mireaDB(MySQL 기반)와 연동하여 실시간 데이터 CRUD 기능을 구성했습니다.
페이징 처리, 댓글 기능, 사용자 인증 등을 구현하며 실전 데이터베이스 연동 경험을 쌓았습니다.
