자기소개

안녕하세요! 개발자 지망생 김도현입니다.
C++, Java, HTML/CSS, Python을 다룰 수 있으며 현재도 계속해서 배우고 있습니다.
웹 프로그래밍은 생활코딩 유튜브 강의를 참고해 기초부터 직접 실습하며 학습해 왔습니다.
게임을 무척 좋아하고, 특히 로스트아크와 오버워치를 오래 즐겼습니다.
오버워치는 한때 프로게이머를 꿈꾸며 전국대회 4강까지 올라간 경험도 있습니다.
최근에는 Discord 플랫폼에 관심을 가져 Python 기반의 노래봇을 개발해보고 있으며,
이를 통해 비동기 프로그래밍과 API 활용 경험도 쌓고 있습니다.

학력

군산대학교 컴퓨터정보공학과

경험 및 활동

  • 웹 게시판 프로젝트에서 mireaDB를 연동하여 데이터베이스 기반 CRUD 기능 구현
  • Discord 봇 개발을 통해 비동기 처리와 외부 API 연동 경험
  • 게임 개발 프로젝트에서 Unity를 활용한 AI 몬스터 행동 구현

사용 가능한 언어

포트폴리오

Unity 몬스터 AI 시뮬레이션

Unity의 NavMeshAgent를 활용해, 몬스터가 시야각(FOV) 기반으로 플레이어를 추적하고
랜덤 이동을 반복하는 간단한 AI를 구현했습니다.

Python Discord 노래봇

Python의 discord.py 라이브러리를 사용해, 사용자가 음성 채널에서 명령어로 음악을 재생할 수 있는
Discord 노래봇을 개발 중입니다. YouTube API와 FFmpeg 등을 연동하여 스트리밍 기능을 구현했습니다.
